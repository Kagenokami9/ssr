# System Flow1

## วัตถุประสงค์
สร้างประสบการณ์ เอฟดฟคสำหรับพิธีเปิดงาน โดยใช้ iPad เป็นอุปกรณ์เริ่มต้น และแสดงเอฟเฟกต์บนจอ LED ขนาดใหญ่

---

## ลำดับการทำงาน

### 1. เริ่มต้น
- ผู้อำนวยการเดินมาที่ iPad
- หน้าเว็บแสดงปุ่ม *Start*

### 2. เริ่มสแกน
- ผู้อำนวยการกดปุ่ม *Start*
- หน้าเว็บเปลี่ยนเป็นหน้าสแกนมือ

### 3. จำลองการสแกน
- แสดงข้อความ *"กรุณาวางมือบนหน้าจอ"*
- เมื่อแตะหน้าจอ ระบบเริ่มแอนิเมชันสแกน
- แอนิเมชันใช้เวลาประมาณ 2–3 วินาที

### 4. ส่งสัญญาณ
- เมื่อสแกนเสร็จ ระบบส่งสัญญาณไปยังคอมพิวเตอร์ที่ควบคุมจอ LED

### 5. แสดงผล
- จอ LED เล่นเอฟเฟกต์ธีมอวกาศ
    - Portal เปิด
    - แสงพุ่ง
    - ดาวระเบิด
    - โลโก้งานปรากฏ

### 6. สิ้นสุด
- ระบบเข้าสู่หน้าพร้อมใช้งานครั้งถัดไป
